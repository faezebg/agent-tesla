# Agent Tesla Project

این ریپازیتوری شامل پروژه عملی و نظری مربوط به تحقیق و بررسی Agent Tesla است. هدف اصلی این پوشه، نگهداری کد نمونه، گزارش‌ها، و مستندات مرجع در قالب سازمان‌یافته است.

## ساختار پروژه

- `بخش عملی/فایل ها/server/server.py`
  - سرور ساده HTTP که درخواست‌های `POST` را دریافت کرده و داده‌ها را در `received.txt` ذخیره می‌کند.
- `بخش عملی/فایل ها/server/received.txt`
  - نمونه داده‌ی دریافت‌شده توسط سرور.
- `بخش نظری/`
  - گزارش نظری و مدارک مرتبط با تحلیل Agent Tesla.
- `منابع/`
  - مقالات و منابع مرجع تخصصی.
- `تعریف و تایید فعالیت.pdf`
  - سند تعریف و تایید فعالیت پروژه.

## اجرای سرور

1. نصب Python 3
2. باز کردن ترمینال در مسیر `بخش عملی/فایل ها/server`
3. اجرای دستور:

```powershell
python server.py
```

4. سرور روی پورت `8000` اجرا می‌شود.

## نکات

- این ریپازیتوری برای نسخه‌بندی Git آماده شده است.
- فایل‌های تنظیمات IDE در `.idea/` در گیت رد می‌شوند.
