## 📖 توضیح تفصیلی محتویات

### 1️⃣ **بخش عملی (Practical Section)**

#### 📌 سرور HTTP (`server.py`)
**هدف:** شبیه‌سازی یک سرور برای دریافت داده‌های ارسالی از Agent Tesla

**ویژگی‌ها:**
- گوش‌دهندگی بر روی پورت **8000**
- پذیرش درخواست‌های `POST`
- ذخیره‌سازی خودکار داده‌های دریافت‌شده
- پاسخ `200 OK` به درخواست‌های موفق

**کد:**
```python
from http.server import BaseHTTPRequestHandler, HTTPServer

class Handler(BaseHTTPRequestHandler):
    def do_POST(self):
        length = int(self.headers['Content-Length'])
        data = self.rfile.read(length)
        
        with open("received.txt", "wb") as f:
            f.write(data)
        
        self.send_response(200)
        self.end_headers()
        self.wfile.write(b"OK")

server = HTTPServer(("0.0.0.0", 8000), Handler)
print("Server running on port 8000...")
server.serve_forever()
```

#### 📄 نمونه داده‌ها (`received.txt`)
حاوی نمونه‌های واقعی (یا شبیه‌سازی‌شده) از داده‌هایی که توسط Agent Tesla ارسال می‌شود:
- تاریخ و زمان اجرا
- اطلاعات کاربری (Username)
- نام رایانه (Computer Name)
